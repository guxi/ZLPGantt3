数据文件格式

注： 数据文件可以是JSON文件(如下格式), 或 Excel表(参见例表)

            
             {
              "name": "减速器加工 3-2",
              "machine": [
                "机床1", "机床2", "磨床"
              ],
              "part": [
                "壳", "架", "颈"
              ],
              "item": [
                [ 0, 0, "切角0.5", 0.0, 4.8 ],
                  ...
                [  ...  ]          
              }
  • 注意字符串都需加上""
  • item项5个数据的含义分别是:

    工位号 , 工件号 , 工艺名称 , 开始时间 , 结束时间
