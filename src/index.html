文本数据文件格式

                
                 {
                  "name": "减速器加工 3-2",
                  "machine": [
                    "机床1", "机床2", "磨床"
                  ],
                  "part": [
                    "壳", "架", "颈"
                  ],
                  "item": [
                    [ 0, 0, "切角0.5", 0.0, 4.8 ],
                      ...
                    [  ...  ]          
                  }
  • 文本数据文件最好是JSON文件, 也可用".txt", 字符集请用utf_8
  • 数据中的字符串都需加上""
  • item项5个数据的含义分别是:

    工位号 , 工件号 , 工艺名称 , 开始时间 , 结束时间

    其中工位号、工件号分别是“machine”和“part”的由0开始的序号
